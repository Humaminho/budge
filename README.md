-   **Node.js**: Version 18.17.0 or later
-   **npm**: Version 8.0.0 or later (comes with Node.js)
-   **Git**: For version control

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/budge-landing-page.git
    cd budge-landing-page
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```

3. **Start the development server**

    ```bash
    npm run dev
    ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

-   `npm run dev` - Start the development server
-   `npm run build` - Build the application for production

### Customization

1. **Content**: Update files in `/src/data/` to modify text, features, and other content
2. **Styling**: Edit `globals.css` and Tailwind classes for visual changes
3. **Components**: Modify React components in `/src/components/` for functionality
4. **Site Details**: Update `src/data/siteDetails.ts` for metadata and branding
